# Maroowell Design System

- 브랜드 성격: 물류 운영, 정산, 스케줄, 실무 데이터 관리
- 우선순위: 가독성 > 속도감 > 감성 디자인
- 기본 테마: 다크 운영툴
- 이미지 복사 테마: 흰 배경 보고서형
- 표 스타일: 촘촘하지만 구분선 명확
- 버튼 체계: 조회/저장/삭제/복사/초기화 색상 고정
- 숫자 데이터 : 우측 정렬
- 텍스트 데이터 : 가운데 정렬
- 라우트/ID: monospace 또는 강한 가독성
- 오류/경고: 즉시 눈에 띄게
- 모바일 페이지 : 모든 페이지는 모바일 폰에서도 볼수 있게 메뉴를 접는 기능 필수.
