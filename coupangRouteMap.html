마루웰 라우트 편집기
캠프 + 라우트 코드 기반 · Supabase / Cloudflare route API 사용
• GET은 기본 prefix 조회로 하위 노선까지 표시합니다.
• camp만 입력하고 GET하면 해당 camp 전체 노선을 표시합니다.
• 지도에서 폴리곤 클릭 → 해당 코드가 편집 대상(녹색)으로 전환됩니다.
• 구역 추가: 같은 코드에 폴리곤을 여러 개 추가(떨어진 구역 지원).
• 저장: 편집 대상 폴리곤(들)을 DB에 저장합니다. (구역 2개 이상이면 다중 배열 저장)
• 지도 정렬: 편집 대상이 있으면 그 노선만 정렬 / 없으면 현재 표시된(prefix 그룹) 전체 정렬 후 자동 저장.
• 라벨 위치: 중심점이 아니라 polylabel(폴리곤 내부 최적)로 잡습니다.
DB 캠프명 수정
• 예: 산격MB로 잘못 등록된 경우 → 입력에 대구3 후 “camp 전체 이관”
• 선택 노선 수정은 현재 편집 대상으로 전환된 노선에만 적용됩니다.
